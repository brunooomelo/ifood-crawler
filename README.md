# Ifood Crawler

> crawler all restaurant name in state


### How to use
  - Clone this project
  - run `npm install`
  - and run `node index.js [state]` :: [default='`sao-paulo-sp`']
  
### How to get state name in ifood
  ![GET state name in ifood](image/get-state.gif)
